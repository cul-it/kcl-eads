These arbitration files are coded to indicate what is included in the file: TR = Transcript(s) B = Brief EX = Exhibit(s) AN = Arbitrator's Notes RC = Routine Correspondence A = Agreement SD = Supporting Document(s)
Access to the collections in the Kheel Center is restricted. Please contact a reference archivist for access to these materials.
