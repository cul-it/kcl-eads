Title:

Abstract:
This is a collection of photographs from Buffalo Labour Studies, BLS Graduates, Career Development, Supervisory Studies, some action shots and miscellaneous ILR images as well as some from the jazz event for the 50th Anniversary reception.
Creator:
Quanitities:
3.5 cubic feet
Language:
Collection material in English
This is a collection of photographs from Buffalo Labour Studies, BLS Graduates, Career Development, Supervisory Studies, some action shots and miscellaneous ILR images as well as some from the jazz event for the 50th Anniversary reception.
Access to the collections in the Kheel Center is restricted. Please contact a reference archivist for access to these materials.
